# Build Todo App Example

Build a simple command-line todo application.

## Requirements

Create a CLI todo app with these features:

1. Add tasks: `node app.js add "Buy groceries"`
2. List tasks: `node app.js list`
3. Complete task: `node app.js done 1`
4. Delete task: `node app.js delete 1`
5. Save tasks to `todos.json`

## Task

Build the complete `app.js` file that implements all these features.

## Testing

```bash
node app.js add "Task 1"
node app.js add "Task 2"
node app.js list
node app.js done 1
node app.js list
```
